## Herramientas previstas

- Cinta métrica y regla metálica.
- Escuadra.
- Cúter con hojas nuevas y base de corte.
- Taladro/atornillador para MDF, solo con guía y protección.
- Sargentos pequeños.
- Soldador y termocontraíble para iluminación, si se incorpora LED.
- Multímetro para verificar alimentación.
